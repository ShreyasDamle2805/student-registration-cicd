# 🎓 Student Registration Portal

A simple and responsive PHP-based Student Registration Portal that allows students to register, log in securely, and access their dashboard. The application uses MariaDB for data storage and follows secure authentication practices with password hashing.

## ✨ Features

- Student Registration
- Secure Login & Logout
- Student Dashboard
- Password Hashing
- Session-based Authentication
- MariaDB Database Integration
- Responsive User Interface

## 🛠️ Tech Stack

- PHP
- HTML5
- CSS3
- JavaScript
- MariaDB (MySQL)
- Apache HTTP Server

## 🚀 Getting Started

1. Clone the repository.
2. Configure your database credentials in `db.php`.
3. Import the required database schema.
4. Place the project inside your web server's root directory.
5. Start Apache and MariaDB.
6. Open the application in your browser.
